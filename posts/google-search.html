Βεβαιωθήτε ότι το blog σας είναι ορατό σε όλους.
Εννοείται πως στις βασικές ρυθμίσεις έχετε επιλέξει το blog σας να είναι δημόσιο, να είναι ορατό στις μηχανές αναζήτησης και καταχωρημένο στον blogger. Δείτε αυτές τις 3 ρυθμίσεις που τις έχω επισημάνει σε ένα screenshout εδώ.

Βάλτε σωστά Metatags μέσα στο πρότυπο.
Θα χρειαστεί να βάλετε τα απαραίτητα Meta-tags μέσα στην ετικέτα <head> για να βοηθήσετε τις μηχανές αναζήτησης να εντοπίσουν ευκολότερα το ιστολόγιό σας. Εάν ενεργοποιήσετε την περιγραφή του blogger δεν χρειάζεται να βάλετε το meta-tag Meta Description μέσα στην ετικέτα <head>. Δείτε εδώ πώς θα βάλετε σωστά τα meta-tags.

Δώστε προσοχή να υπάρχουν τα παρακάτω Metatags για την γλώσσα και την Χώρα.
<meta content='text/html; charset=utf-8' http-equiv='Content-Type'/>
<meta content='el-gr, en-us' http-equiv='content-language'/>
<meta content='Greece' name='geo.country'/>
<meta content='Greece' name='geo.placename'/>

Μην βάζετε εικόνα αντί του τίτλου.
Μην βάζετε εικόνα αντί του τίτλου του blog γιατί δεν θα διαβάζουν οι μηχανές αναζήτησης τον τίτλο του blog σας. Μπορείτε να βάλετε την εικόνα πίσω από τον τίτλο ή δίπλα στον τίτλο σαν gadget αφού χωρίσετε την κεφαλίδα σε 2 μέρη. Αυτό για να το κάνετε δείτε αυτήν την ανάρτηση.

Μη χρησιμοποιείτε δύσκολα αναγνωρίσιμες γραμματοσειρές.
Για να είστε 100% σίγουροι ότι οι μηχανές αναζήτησεις θα διαβάσουν το περιεχόμενό σας καλό θα ήταν να χρησιμοποιήτε γραμματοσειρά Arial που είναι η πιο αναγνωρίσιμη από όλες και όχι άλλες σπανιότερες που μπορεί να δημιουργήσουν πρόβλημα στην ανάγνωση του ελληνικού κειμένου.

Δηλώστε ένα XML Sitemap.
Δηλώστε και ένα XML Sitemap του ιστολογίου σας το οποίο θα πρέπει να υποβάλλετε στο Google Webmaster Tools και στο Bing Webmaster Tools αφού είστε συνδεδεμένοι με τους λογαριασμούς gmail.com και hotmail.com αντίστοιχα.

Κοινοποιήστε τα άρθρα σας στα Sosial Networks.
Αρχίστε να κοινοποιήτε τις αναρτήσεις του blog σας στο facebook, στο Google+ και στο twitter.
Αφού το κάνετε αυτό πατήστε πάνω σε αυτά τα links για να πάρει χτυπήματα το blog σας από τα Sosial Networks και κάντε σε όλα τα άρθρα σας εσείς και ει δυνατόν και οι φίλοι σας like και +1.

Βάλτε την διεύθυνση url του blog σας στην αναζήτηση Google.
Μετά από όλα αυτά βάλτε την διεύθυνση url του blog σας στην αναζήτηση Google και πατήστε enter να δείτε αν βγαίνει πρώτο. Λογικά θα πρέπει σε 1-2 μέρες αφου κάνετε τα παραπάνω να εμφανιστεί. Αφού εμφανιστεί βάλτε τώρα μερικούς τίτλους από κάποιες αναρτήσεις που έχετε κάνει να δείτε σε ποια θέση βγαίνουν. Αν έχετε σωστές ετικέτες και κόσμο να πατάει τα άρθρα σας από τα κοινωνικά δίκτυα τότε οι αναρτήσεις σας και πολύ περισσότερο η αρχική σελίδα του blog σας θα πρέπει να βγαίνουν σε καλές θέσεις εκτός και υπάρχει πολύς ανταγωνισμός σε αυτό το αντικείμενο που ασχολείται το blog σας.
