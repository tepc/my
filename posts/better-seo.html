CODES AND TIPS FOR WEB DESIGNERS

ΣΥΜΒΟΥΛΕΣ ΓΙΑ ΕΝΑ BLOG ΜΕ ΚΑΛΟ SEO, ΕΜΦΑΝΙΣΙΜΟ & ΛΕΙΤΟΥΡΓΙΚΟ
1. Επιλέξτε ένα σύντομο url / domain name.
Ένα σύντομο url / domain name είναι πολύ πιο έυκολο να το θυμάται κάποιος από ένα μακρύτερο. Αυτό το λέω γιατί θα χρειαστεί να πέιτε σε κάποιους μπήτε το blog μου και να του εξηγήτε ότι μετά την πρώτη λέξη έχει παύλα ενώ η τρίτη είναι κολλητά με την δεύτερη. Άριστο είναι να περιέχει μία λέξη χωρίς παύλες και χωρίς αμφιβολία αν γράφεται με h αντί i ή με w αντί ο.


2. Βάλτε λέξεις κλειδιά μέσα στο url / domain name.
Είναι καλό να υπάρχουν λέξεις-κλειδιά μέσα στο url / domain name γιατί αυτό θα βοηθήσει το blog μας να βγαίνει ψηλότερα στην αναζήτηση της google. Όπως για παράδειγμα στο δικό μου url υπάρχει η λέξη κλειδί “widgets”. Αν κάποιος αναζητήσει στο google widgets για blogs και υποθέσουμε ότι υπάρχουν 2 τέτοια blog με τις ίδιες λέξεις κλειδιά σε όλα τα σημεία (τιτλό & υπότιτλο blog, τίτλο ανάρτησης και κείμενο ανάρτησης) θα βγεί ψηλότερα αυτό που έχει επιπλέον την λέξη κλειδί στο url του. Γιαυτό είναι σημαντικό να βάζετε λέξη κλειδί μέσα στο url του blog σας.  

3. Βάλτε λέξεις κλειδιά μέσα στον Τίτλο.
Ο τίτλος ενός blog (title tag) θα πρέπει να περιέχει μία, δύο ή τρείς λέξεις κλειδιά για να πετύχει και με αυτό τον τρόπο καλύτερη θέση στη αναζήτηση της google. Προσπαθήστε ο τίτλος να περιγράφει εν συντομία το αντικείμενο που ασχολείται το blog και να είναι περίπου έως 70 χαρακτήρες.

4. Βάλτε λέξεις κλειδιά μέσα στον υπότιτλο.
Ο υπότιτλος καλό είναι και αυτός να περιέχει λέξεις κλειδιά για λόγους καλύτερου SEO όπως προανέφερα και πιο πάνω απλά εδώ προσέξτε να μην αντιγράψετε ακριβώς τον τίτλο αλλά να ξεκινάτε κάπως διαφορετικά με λέξεις πουν δεν περιέχονται στον τίτλο και μετά να συμπεριλάβετε ακόμα και τις ίδιες λέξεις κλειδιά που περιέχονται και στον τίτλο.

5. Βάλτε όλα τα keywords στην πρώτη παράγραφο.
Προσπαθήστε μέσα στη πρώτη παράγραφο να τοποθετήσετε όλα τα κυρια keywords του άρθρου σας για να ανιχνευθούν ευκολότερα από τις μηχανές αναζήτησης.

6. Βάλτε τις σωστές ετικέτες.
Χρησιμοποιήστε μία γενική ετικέτα και 3-4 ειδικές για την κάθε ανάρτησή σας. Για παράδειγμα αν ο τίτλος της ανάρτησης είναι Θεραπεύτε τον καρκίνο χωρίς χημειοθεραπεία. Τότε η γενική ετικέτα θα πρέπει να είναι, ΑΝΤΙΚΑΡΚΙΝΙΚΑ ΑΡΘΡΑ και οι ειδικές ετικέτες θα πρέπει να είναι, θεραπεία καρκίνου χωρίς χημειοθεραπεία, φυσικές θεραπείες καρκίνου, εναλλακτικές θεραπείες καρκίνου. Με αυτόν τον τρόπο θα πετύχετε να βγαίνουν οι αναρτήσεις σας στην αναζήτηση του google αρκετά ψηλά.

7. Αποφύγετε τις υπεράριθμες κατηγορίες στις αναρτήσεις σας.
Φροντίστε κάθε ανάρτηση να ανήκει σε μία κατηγορία (γενική ετικέτα) γιατί αν ανήκει σε περισσότερες θα έχετε πρόβλημα στο να φτιάξετε μία σελίδα περιεχομένων για τις αναρτήσεις που περιέχει το blog σας, αφού θα παρουσιάζονται σε 2 και 3 κατηγορίες θεμάτων οι συγκεκριμένες αναρτήσεις και θα χρειάζεται περισσότερο χρόνο ο αναγνώστης για να βρεί αυτό που θέλει.

8. Ενεργοποιήστε την περιγραφή του Blogger.
Ενεργοποιήστε την περιγραφή για κάθε ανάρτηση που σας προσφέρει ο blogger στις Ρυθμίσεις / Προτιμήσεις αναζήτησης / Μετα-ετικέτες Περιγραφή. Δείτε εδώ. Όταν την ενεργοποιήσετε θα μπορείτε όταν κάνετε μία ανάρτηση να βάζετε εκτός από τις ετικέτες και περιγραφή αναζήτησης (δείτε εδώ) και έτσι θα βγαίνει η ανάρτηση σε καλύτερη θέση στην αναζήτηση google.

9. Βάλτε σωστά Metatags μέσα στο πρότυπο.
Θα χρειαστεί να βάλετε τα απαραίτητα Meta-tags μέσα στην ετικέτα <head> για να βοηθήσετε τις μηχανές αναζήτησης να εντοπίσουν ευκολότερα το ιστολόγιό σας. Εάν ενεργοποιήσετε την περιγραφή του blogger δεν χρειάζεται να βάλετε το meta-tag Meta Description μέσα στην ετικέτα <head>. Δείτε εδώ πώς θα βάλετε σωστά τα meta-tags.

10. Φροντίστε να αλλάξετε τα permalinks.
O blogger σας δίνει την δυνατότητα να επιλέξετε το permalink της κάθε ανάρτησης. Αυτό θα το βρείτε λίγο πιο κάτω από τις ετικέτες δείτε εδώ. Φροντίστε να δώσετε ένα permalink που να περιέχει λέξεις κλειδιά από τον τίτλο της ανάρτησης αλλά και της περιγραφής ALT της εικόνας ανάρτησης για να μην έχετε και πρόβλημα στην εμφάνηση της σωστής εικόνας όταν κοινοποιήτε την αναρτηση αυτή στο facebook. Εννοείται ότι αυτό θα βοηθείσει την ανάρτηση να βγαίνει ψηλότερα στην αναζήτηση της google. Χρησιμοποιήστε παύλες ( – ) στα permalinks και όχι άλλους χαρακτήρες για τον διαχωρισμό των λέξεων.

11. Βάζετε ετικέτες  ALT στις εικόνες.
Για να δώσουμε στις μηχανές αναζήτησης να καταλάβουν τι αφορά μια εικόνα που υπάρχει στο blog σας ορίζουμε τις ετικέτες Alt (σε κάθε περίπτωση) και Title (αν η εικόνα είναι και σύνδεσμος/link).

12. Δηλώστε ένα XML Sitemap του ιστολογίου σας.
Το Sitemap είναι ο καλύτερος τρόπος για να δηλώσετε τη δομή και τον αριθμό των συνδέσμων της ιστοσελίδας σας. Μπορείτε να βρείτε διάφορα εργαλεία στο ιδαδίκτυο που θα φτιάξουν το XML sitemap του blog σας, το οποίο στη συνέχεια θα πρέπει να υποβάλλετε στο Google Webmaster Tools και στο Bing Webmaster Tools.

13. Μην σβήνετε τις αναρτήσεις που δεν χρειάζεστε. Αντικαταστήστε τες με νέες.
Οι μηχανές αναζήτησης αντιπαθούν τις σελίδες 404 (σφάλματος), οπότε θα πρέπει να χρησιμοποιείτε σωστά 301 redirects (ανακατευθύνση), ή καλύτερα να μην σβήνετε τις αναρτήσεις που δεν θέλετε πια να υπάρχουν στο blog σας αλλά γράψτε μέσα τους νέο περιεχόμενο με νέο τίτλο ανάρτησης και νέες ετικέτες.

14. Αποφύγετε την χρήση Frames.
Αποφύγετε τη χρήση των frames, είναι απαρχαιωμένη τακτική και ενδέχεται να επηρρεάζουν αρνητικά την σάρωση της ιστοσελίδας σας από τις μηχανές αναζήτησης.

15. Αποφύγετε την χρήση Flash.
Αν χρησιμοποιείτε flash (συνίσταται να αποφεύγεται) τουλάχιστον εισάγετε κείμενο καθώς και τίτλους/περιγραφές και ετικέτες επικεφαλίδας.

16. Χρησιμοποιήστε on line seo tools.
Χρησιμοποιήστε online seo tools που σας βοηθάνε να εντοπίσετε και να αντιμετωπίσετε τα πρόβλημα της ιστοσελίδας σας. Δείτε εδώ τα καλύτερα on line seo tools.

17. Καταχωρήστε το blog σας σε καταλόγους.
Η καταχώρηση σε καταλόγους είναι ένας παλιός και παραδοσιακός τρόπος. Ωστόσο, μην κάνετε πολλές καταχωρήσεις και μαζικά. Αρκεστείτε στη καταχώρηση σε σημαντικούς καταλόγους όπως στη GOOGLE και στη BING .

18. Γράφτε για το ιστολόγιό σας στα Forum.
Συμμετάσχετε σε διάφορα forum και μοιραστείτε συνδέσμους (προς το blog σας μαζί με χρήσιμες παραπομπές για τους υπόλοιπους χρήστες) μαζί με τις απαντήσεις. Με αυτόν τον τρόπο μπορείτε να αυξήσετε γρήγορα τον αριθμό των επισκεπτών σας.

19. Γράψτε ενδιαφέροντα και επίκαιρα άρθρα σε καταλόγους άρθρων ή σε φιλικά blogs.
Η συγγραφή άρθρων μπορεί να αποφέρει πολλά καλά links προς το blog σας. Υπάρχουν πολλoί κατάλογοι άρθρων για να υποβάλετε άρθρα με συμβουλές ή πληροφορίες σχετικές με το περιεχόμενο του blog σας. Ακόμα καλύτερα αν κάποιο άλλο blog προσφέρεται να φιλοξενίσει άρθρα σας στα οποία φυσικά θα υπάρχει παραπομπή με ενεργό link προς το blog σας.

20. Σχολιάστε σε άλλα blogs αναφέροντας το blog σας.
Μπορείτε να σχολιάσετε ή να απαντήσετε σε σχόλια που γίνονται σε άλλα blogs με παραπομπή σε ένα άρθρο που βρίσκεται στο blog σας, το οποίο και αυτό μπορεί να αποφέρει επιπλέον links προς το blog σας. Υπάρχουν πολλoί κατάλογοι άρθρων για να υποβάλετε άρθρα με συμβουλές ή πληροφορίες σχετικές με το περιεχόμενο του site σας.

21. Βρήτε και άλλους συγγραφείς.
Δώστε τη δυνατότητα στο κόσμο να ανεβάσει τα δικά του άρθρα. Θα εκπλαγείτε από την επισκεψιμότητα που μπορεί να σας προσελκύσει.

22. Προσθέτετε συχνά νέες αναρτήσεις.
Το περιεχόμενο (κείμενο) είναι ένας βασικός παράγοντας για τη βελτιστοποίηση μιας σελίδας. Οι μηχανές λατρεύουν το φρέσκο περιεχόμενο! Προσθέτετε ή αλλάζετε συχνά κείμενο στις σελίδες σας.

23. Γράψτε μοναδικό περιεχόμενο.
Το περιεχόμενο σας πρέπει να είναι μοναδικό, χωρίς αντιγραφές και σωστά δομημένο. Αποφύγετε την κακή χρήση του συντακτικού και τα ορθογραφικά λάθη. Κρατήστε ανανεωμένο το περιεχόμενο σας καθώς με αυτό το τρόπο παρουσιάζεται ενδιαφέρον προς τους χρήστες.

24. Φτιάξτε σελίδες στα κοινωνικά δίκτυα.
Φροντίστε να φτιάξετε σελίδες στα Social Networks όπως το Facebook, Twitter, Google Plus, Youtube και το Pinterest. Δημοσιεύοντας εκεί τις αναρτήσεις του blog σας θα πάρατε πολλά ποιοτικά χτυπήματα προς το blog σας από Social Networks με υψηλό Google rank, το οποίο θα ανεβάσει το blog σας ψηλότερα στην κατάταξη Google rank και Alexa rank.

25. Βάλτε στοιχεία επικοινωνίας.
Η Google εκτιμάει της σελίδες με ξεκάθαρα στοιχεια επικοινωνίας. Ο κώδικας της ιστοσελίδας σας πρέπει να είναι σωστός, δείτε τα πρότυπα του W3C Validation και συμμορφωθείτε με αυτά.

26. Μειώστε τον χρόνο φόρτωσης του blog σας.
Το blog σας πρέπει να είναι γρήγορο, φροντίστε το loading time να μην είναι μεγάλο, αν αντιμετωπίζετε προβλήματα ταχύτητας ίσως πρέπει να εγκαταστήσετε ένα Cache plugin η να συμπιέσετε το μέγεθος των εικόνων, ή να αφαιρέσετε κάποια χρονοβόρα widgets. Σε αυτήν σας την προσπάθεια θα σας βοηθήσει ένα εργαλείο που μετράει τον χρόνο φόρτωσης και γενικώς τι καθυστερεί το blog σας. Είναι το Pingdom tools.

27. Κάντε το ιστολόγιό σας φιλικό προς τον χρήστη.
Το Site σας πρέπει να είναι φιλικό προς τον χρήστη, όσο περισσότερο χρόνο παραμένει, τόσο το καλύτερο για το SEO σας. Προτιμήστε σκουρόχρωμο πρότυπο είναι πιο ξεκούραστο για τα μάτια του αναγνώστη. Προτιμήστε 2 πλευρικές μπάρες αντί για μία θα σας βοηθήσουν να μοιράσετε τα gadgets καλύτερα να βάλετε π.χ. αυτά που παραπέμπουν εντός του blog στην αριστερή μπάρα και τα βοηθητικά, και αυτά που παραπέμπουν εκτός του blog σας στην δεξιά.
Αξιοποιήστε και το υποσέλιδο και εκεί μπορείτε να βάλετε κάποια gadgets.

28. Βάλτε στο πλάι τα πιο σημαντικά άρθρα.
Καλό θα είναι να διασυνδέσετε το περιεχόμενό σας, κάνοντας την πλοήγηση πιο εύκολη και χρήσιμη. Μπορείτε να βάλετε τα πιο σημαντικά άρθρα στην αριστερή πλαϊνή μπάρα, έτσι κάνοντας scrolling ο αναγνώστης μπορεί να βρεί κάποιο άρθρο ενδιαφέρον, να μπεί να το διαβάσει και να μείνει περισσότερο χρόνο στο blog σας και να έχετε περισσότερες προβολές σελίδων που είναι και τα ζητούμενα.

29. Βάλτε σχετικές εικόνες και βίντεο μέσα στις αναρτήσεις σας.
Να συμπεριλαμβάνεται εικόνες και video σχετικά με το κείμενο σας. Βοηθούν τον αναγνώστη να κατανοήσει καλύτερα τι γράφετε στο κείμενό σας αν υπάρχουν εικόνες ή βίντεο που δείχνουν τι εννοείτε, και φυσικά αν δούν ένα βίντεο εντός της ανάρτησης αυτό σημαίνει αυτόματα περισσότερος χρόνος παραμονής στο blog σας και άρα άνοδος στο SEO του.

30. Χρησιμοποιείστε το μικρότερο δυνατό μέγεθος για τις εικόνες σας.
Καλο είναι να μη ξεπερνάνε τα 20 kb για την εικόνα της ανάρτησης που βρίσκεται πρίν το διαβάστε περισσότερα και για τα εικονίδια που τυχόν έχετε στις πλευρικές μπάρες να είναι μέχρι 5kb το πολύ και διαστάσεων 90X60. Το ιδανικό μέγεθος της εικόνας ανάρτησης είναι 450Χ300 και αναλογία 3:2 για όλες, γιατί αργότερα μπορεί να θελήσετε να βάλετε slider, να μην σας κάνει παραμόρφωση σε κάποιες που δεν έχουν αυτήν την επικρατέστερη αναλογία. Κάτω από το διαβάστε περισσότερα μπορείτε να βάλετε μεγαλύτερα μεγέθη εικόνων. Συμπίεση εικόνων κάνετε με το πρόγραμμα ifran view. Δείτε εδώ πως κόβετε τις εικόνες στην ίδια αναλογία και πως συμπιέζετε το μέγεθος τους σε ΜΒ ή Kb.

31. Φροντίστε ότι το blog σας είναι προσβάσιμο από Smartphones και Tablets.
Δείτε πώς εμφανίζετε το blog σας στα κινητά και στα tablets και προσαρμόστε την εμφανισή του από την προσρμογή για mobile συσκευές που έχει ο blogger ετσι ώστε να είναι εύκολη η ανάγνωσή του και από αυτές τις συσκευές.

32. Επιλέξτε μεγαλύτερο πλάτος ιστολογίου.
Τα επικρατέστερα πλάτη για blogs είναι 1000px, 1280px, 1360px και 1500px. Όσο πιο πλατύ είναι ένα blog τόσο πιο πρακτικό είναι και λειτουργικό (εννοείται όχι πάνω από 1500px). Το καλύτερο θα ήταν να εκμεταλευτήτε το τελευταίο πλάτος των 1500px που σας επιτρέπει ο blogger για να μπορείτε να έχετε μεγάλο πλάτος κεντρικής ανάρτησης, το οποίο θα σας χρειαστεί για να φτιάξετε τις εξής σελίδες : Αρχειοθήκη αναρτήσεων με 2 στήλες, photo gallery, video gallery, ενσωμάτωση ολόκληρων sites όπως forum, google drive, dropbox, box, image shack, photobucket, google+ gallery και άλλα πολλά τα οποία χρειάζονται πλάτος ανάρτησης πάνω από 950px. Το μειονέκτημα είναι πως οι μισοί περίπου αναγνώστες που έχουν φθηνά laptops ή tablets με αναλύσεις 1360px και 1280px θα χάνουν ένα μέρος από την 2 πλευρική μπάρα περίπου 140px με 220px, αλλά εσείς μπορείτε να φροντίσετε η δεξιά πλευρική μπάρα να έχει δευτερεύουσας σημασίας gadgets και πληροφορίες που δεν επειρεάζουν την παραμονή του επισκέπτη στο blog σας όπως στατιστικά ιστολογίου, ώρα, ημερομηνία, καιρός, facebook like box, google followers box, λίστα φιλικών ιστολογίων και λοιποί σύνδεσμοι που οδηγούν εκτός του blog σας. Αυτό έχω εφαρμώσει και εγώ σε αυτό το ιστολόγιο που διαβάζετε και απολαμβάνω την ευρυχωρία και την λειτουργικότητα χωρίς οι επισκέπτες να χάσουν κάτι σημαντικό.
