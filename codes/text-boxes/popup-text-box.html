ΑΝΑΔΥΟΜΕΝΟ TEXT BOX ΠΟΥ ΑΝΟΙΓΕΙ ME BUTTON
Το παρακάτω text box ανοίγει σε όποια στατική θέση του ιστολογίου σας θέλετε πατώντας πάνω σε ένα button. Η θέση του text box αλλάζει εκεί που γράφει position:fixed; top:0px;left:0px; και οι διαστάσεις του text box εκεί που γράφει width:490px; height:auto; Αντικαταστήστε τους κίτρινους τίτλους με τους δικούς σας. Τα χρώματα μπορείτε να τα αλλάξετε αλλάζοντας τον 6ψήφιο αριθμό που βρίσκεται μετά το σύμβολο της δίεσης #. Ο κώδικας μπορεί να μπεί απλά σαν gadget ή ακόμα και μέσα στο html της ανάρτησής σας.
